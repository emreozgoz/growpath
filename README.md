## 🎯 Vision

GrowPath empowers learners to achieve their goals through AI-powered personalized learning paths. Whether you're learning a new programming language, mastering a framework, or developing any skill, GrowPath creates a customized weekly plan that fits your schedule and experience level.

## ✨ Key Features

- **🤖 AI-Powered Planning**: Get personalized learning plans generated by advanced AI (GPT-3.5 for free users, GPT-4 for premium)
- **📊 6-Step Onboarding**: Intuitive stepper interface to capture your learning preferences
- **🎯 Goal-Oriented**: Set clear objectives and track your progress
- **📅 Flexible Scheduling**: Plans adapt to your weekly time commitment
- **📈 Progress Tracking**: Monitor your learning journey with detailed analytics
- **🌱 Green Theme**: Beautiful, calming interface designed for focus

## 🚀 Tech Stack

### Frontend
- **Framework**: Next.js 14 (App Router, React Server Components)
- **Language**: TypeScript 5.0
- **Styling**: Tailwind CSS 3.4 (Green theme: #10B981 Emerald)
- **UI Components**: Radix UI (Accessible, unstyled primitives)
- **State Management**: Zustand + React Query (Server state caching)
- **Forms**: React Hook Form + Zod validation

### Backend
- **Runtime**: Node.js 20 LTS
- **Framework**: Express.js
- **Database**: PostgreSQL 16 (Primary data store)
- **Cache**: Redis 7 (Session + AI response caching)
- **ORM**: Prisma (Type-safe database access)
- **Authentication**: NextAuth.js (OAuth + Email/Password)

### AI & External Services
- **AI Provider**: OpenAI API (GPT-3.5-turbo / GPT-4-turbo)
- **Payments**: Stripe (Subscription management)
- **Email**: Resend (Transactional emails)

### Infrastructure
- **Hosting**: Vercel (Frontend + Edge Functions)
- **Database**: Supabase / Railway (Managed PostgreSQL)
- **Cache**: Upstash Redis (Serverless Redis)
- **Monitoring**: Sentry (Error tracking) + Vercel Analytics

## 🏃 Getting Started

### Prerequisites
- Node.js 20 LTS or higher
- PostgreSQL 16
- Redis 7
- OpenAI API key

### Installation

```bash
# Clone the repository
git clone https://github.com/emreozgoz/growpath.git
cd growpath

# Install frontend dependencies
cd frontend
npm install

# Install backend dependencies
cd ../backend
npm install

# Set up environment variables
cp .env.example .env
# Edit .env with your configuration

# Run database migrations
npx prisma migrate dev

# Start development servers
npm run dev
```
